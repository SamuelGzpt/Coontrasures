PROJECT DEPENDENCIES for COONTRASURES

This project uses Node.js (npm). Below are the main dependencies for both Client and Server.

=========================================
CLIENT (Frontend)
Located in: ./
Install with: npm install
=========================================
Dependencies:
- gsap: ^3.14.2 (Animations)
- lucide-react: ^0.562.0 (Icons)
- react: ^19.2.0 (UI Library)
- react-dom: ^19.2.0
- react-router-dom: ^7.11.0 (Routing)

Dev Dependencies:
- vite: ^7.2.4 (Build Tool)
- typescript: ~5.9.3
- eslint: ^9.39.1
- @vitejs/plugin-react: ^5.1.1

=========================================
SERVER (Backend)
Located in: ./server
Install with: cd server && npm install
=========================================
Dependencies:
- cors: ^2.8.5 (Cross-Origin Resource Sharing)
- dotenv: ^17.2.3 (Environment Variables)
- express: ^5.2.1 (Web Framework)
- express-rate-limit: ^8.2.1 (Security)
- helmet: ^8.1.0 (Security)
- multer: ^2.0.2 (File Uploads)
- pg: ^8.17.2 (PostgreSQL Client)

=========================================
SYSTEM REQUIREMENTS
=========================================
- Node.js (v18 or higher recommended)
- PostgreSQL Database
